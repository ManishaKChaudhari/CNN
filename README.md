# CNN
CNN stands for convolutional neural network which is a deep learning algorithm.
Dependencies: 
Jupyter notebook,
 Tensorflow 1.10,
 Python 3.6,
 Matplotlib,
 Seaborn,
 Scikit-Learn,
 Pandas,
 Numpy.
Using above dependencies we perform cnn algorithm to classify between image that whether it is cat or dog.
